📝 노드 설명

  • extract_metadata: 사용자 쿼리에서 도메인(카드/대출/보험)과 대화 턴 수 추출
  • retrieve_examples: Pinecone/ChromaDB에서 유사한 상담 예시 검색 (Few-shot)
  • route_by_environment: 환경 감지 후 적절한 모델로 라우팅
    • GPU 감지 시 → vllm_counselor
    • Mac/CPU 감지 시 → ollama_counselor
  • vllm_counselor: vLLM으로 답변 생성 (Prefix Caching으로 멀티턴 최적화)
  • ollama_counselor: Ollama로 답변 생성 (KV Cache로 멀티턴 최적화)

✨ 주요 특징

  • 조건부 라우팅: add_conditional_edges()로 환경별 분기 처리
  • 환경별 최적화:
    • GPU: vLLM + Prefix Caching (고성능)
    • Mac/CPU: Ollama + KV Cache (경량)
  • 통합 프롬프트: 두 경로 모두 동일한 build_qa_prompt() 사용
  • 멀티턴 대화: 슬라이딩 윈도우로 최근 8-10턴 유지
